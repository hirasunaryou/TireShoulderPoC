# Codex向け 次の依頼テンプレート

以下をそのままCodexへ投げて、作り込んでいく想定です。

---

このSwiftUI iOSアプリは、2本のUSDZからタイヤショルダ部のプロファイル差分を比較するPoCです。

現状:
- Filesから new.usdz / used.usdz を読み込む
- USDZテクスチャから青/赤のマスクを抽出する
- 青点群で PCA + ICP による位置合わせを行う
- 赤点群を2Dプロファイル化し、差分グラフとCSVを出す

次の改善を行ってください。

優先順位:
1. コンパイルエラーがあれば修正
2. iPhone実機で落ちる箇所を潰す
3. 色しきい値をUIから調整できるようにする
4. 自動抽出後に青/赤マスクを確認できるデバッグ表示を追加する
5. 解析ログを画面に出す
6. READMEにセットアップ手順を追記する

制約:
- iPhoneローカルのみで完結
- 外部サーバーは使わない
- SwiftUIベースを維持
- まずはPoC優先、過度な設計抽象化は不要

出力してほしいもの:
- 変更後のファイル一覧
- 変更理由
- 実機での確認手順
- 残課題
